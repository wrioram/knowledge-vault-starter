# Standard Operating Procedures

SOPs are step-by-step procedures that tell the assistant exactly what to do for each repeatable vault operation.

## How to use

1. Keep these files in `ops/SOPs/`.
2. Read `ops/INDEX.md` at session start.
3. Follow the relevant SOP when an operation is triggered.

## Included SOPs

| File | Operation |
|---|---|
| SOP-001-ingest.md | Ingest a new external source into the wiki |
| SOP-002-journal-entry.md | Write a journal entry |
| SOP-003-new-contact.md | Create a new contact page |
| SOP-004-health-check.md | Vault health check and lint |
| SOP-005-new-project.md | Start tracking a new project |
| SOP-006-meeting-ingestion.md | Process a meeting transcript or note |
